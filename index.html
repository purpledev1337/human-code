Implementare un layout assegnato come esercitazione pomeridiana Boolean.

  1. Clonazione della repo GitHub su VSC
  2. Guardo e analizzo la struttura del layout
    • Individuo la macro struttura
    • Individuo le parti di contenuto statiche/semplici
    • Individuo le parti di contenuto più complesse come effetti/menu a comparsa/comportamento della pagina
    • Mi soffermo mentalmente sulle parti più complesse e valuto se ho già gli strumenti per replicarle
    • Valuto in generale se e quali parti di layout sono replicabili
  3. Scrivo la macro struttura con del codice di debug
  4. Inserisco tutto il contenuto statico/semplice senza specificare font/colori/margini
  5. Affronto le parti più complesse
    1. Provo a scrivere la soluzione senza considerare font/colori/margini
    2. Controllo funzionamento della sezione > (Se funziona vai al punto 6 altrimenti continua)
    3. Ricerca online di una soluzione riguardo il problema > (Se la trovo torna al punto 5.1 altrimenti continua)
    4. Apertura Ticket su Slack > (Se ok vai al 5.1 altrimenti torna al 5.3)
  6. Replico le parti replicabili inserendo le dovute classi
  7. Personalizzo le varie sezioni con il loro rispettivo font/colore/margine
  8. Ricontrollo generale sul funzionamento di tutte le sezioni > (Se funziona tutto vai avanti altrimenti torna al punto 5)
  9. Controllo del codice in cerca di errori/ripetizioni o ottimizzazione classi/selettori > (Se funziona vai avanti altrimenti torna al punto 8)
  10. PUSH su GitHub
